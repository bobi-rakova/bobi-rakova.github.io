HACKIDEMIA

Hackidemia is a global network that designs workshops and kits enabling kids to use curiosity, play, and empathy to solve global challenges. I have developed content for electronics workshops involving Arduino, Raspberry Pi, sensors, displays and others. I have a great team of more than 20 volunteers in Bulgaria to help me keep the initiative alive and organize regular events.

Some of the workshops include activitines that teach:

  • Basic Machine Learning concepts through training and testing an image recognition model together with the children.
  • Building games in Virtual Reality using the Unity framework and Gear VR
  • Basic ellectoncis concepts exploration by using paper stickers, sensors and microcontrollers and programming them into an interactive circuit using Arduino.
  • Building a meteorological station to monitor air quality, temperature, humidity and other parameters.
